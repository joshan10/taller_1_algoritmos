# Taller 1 - Algoritmos

## Descripción
Proyecto educativo de algoritmos para demostrar conceptos fundamentales de programación.

## Requisitos
- Python 3.8+
- pip (gestor de paquetes)
- colorama


## Menú Principal
El programa ofrece las siguientes opciones:

1. **Algoritmos del 1 al 25** - Implementaciones de bubble sort, merge sort, quick sort
2. **Salir** - Terminar la aplicación

## Uso
```bash
python index.py
```

## Librerías Utilizadas
- `sys` - Manejo de sistema
- `time` - Medición de rendimiento
- `os` - Interacción con el sistema operativo

## Características Principales
- Interfaz en la terminal interactiva mediante `index.html`
- Visualización en tiempo real de algoritmos
